# ✦ Expense Tracker

A clean, minimal expense tracking app built with React and Tailwind CSS.

🔗 **Live Demo**: [haca-py04-expense-tracker.netlify.app](https://haca-py04-expense-tracker.netlify.app/)

---

## Features

- View total expenses for the current month
- Add expenses with category, title, and amount
- Browse recent transactions sorted by date
- Fully responsive — works on mobile and desktop

## Tech Stack

- **React** — component-based UI
- **Tailwind CSS** — utility-first styling
- **Netlify** — hosting and deployment

## Getting Started

```bash
git clone https://github.com/your-username/expense-tracker.git
cd expense-tracker
npm install
npm run dev
```
